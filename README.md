# Introduction

vue2.0 开发环境搭建

---

### *环境搭建*

#### 安装node环境

官网<http://nodejs.cn/>下载最新nodejs并安装

#### 安装webpack

    npm install webpack -g

#### 安装vue脚手架

    npm install vue-cli -g

#### 创建基于webpack模版的新项目

    vue init webpack admin-vue2
    //安装依赖
    cd admin-vue2
    npm install
    npm run dev
